
# RSS Aggregator (Go)

A simple **RSS feed aggregation backend written in Go**.
This project fetches posts from RSS feeds, stores them in a PostgreSQL database, and exposes functionality to manage feeds and users.

The project was built while following the **Boot.dev backend course by Wagslane** and is mainly intended as a **learning exercise** to understand backend development with Go and PostgreSQL.

---

## What I Learned

* Building a backend service in **Go**
* Working with **PostgreSQL**
* Managing **database migrations** using Goose
* Generating type-safe database access code using **SQLC**
* Structuring a backend project
* Fetching and parsing **RSS feeds**

---

## Tech Stack

* Go
* PostgreSQL
* SQLC (type-safe SQL code generation)
* Goose (database migrations)

## Database

PostgreSQL is used as the primary data store.

Schema changes are managed using **Goose migrations**, and SQL queries are compiled into type-safe Go code using **SQLC**.
